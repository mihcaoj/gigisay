# gigisay

## How to run

If you don't have it, [install Go](https://go.dev/dl/).

### Clone the repo
```bash
git clone https://github.com/mihcaoj/gigisay/
```
### Change directory
```bash
cd gigisay
```
### Build binary for your OS
Linux/macOS:
```bash
go build -o gigisay
```
Windows:
```bash
go build -o gigisay.exe
```
### Run it
Linux/macOS:
```bash
./gigisay YOUR TEXT HERE
```
Windows:
```bash
.\gigisay.exe YOUR TEXT HERE
```
